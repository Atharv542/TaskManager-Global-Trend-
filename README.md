# Task Manager – MERN Stack Internship Project

A full-stack Task Management Web Application built as part of a Full Stack Development Internship skill assessment.

This application allows users to register, log in, create tasks, update their status, view task details, edit tasks, and delete tasks. Each user has their own private task list secured with JWT authentication.

##  Features

### Authentication
- User Registration & Login
- JWT based authentication
- Token stored in localStorage
- Logout functionality

### Task Management
- Create tasks
- View all tasks
- View individual task details
- Edit tasks
- Delete tasks
- Task status (Pending / Completed)
- Toggle task completion

### UI/UX
- Dark theme UI
- Responsive design (mobile + desktop)
- Navbar with user name
- Separate pages for Add Task and View Tasks
- Toast notifications (react-hot-toast)
- Empty state handling

##  Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- React Hot Toast

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs

### Deployment
- Frontend: Vercel
- Backend: Render
